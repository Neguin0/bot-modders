# Bot Consultas Modders
Bot para whatsapp para realizar consultas de dados\
usamos uma API Gratuita para fazer as consultas.

## Requisitos
- NodeJS

## Instalação
Clone o repositório
```bash
git clone https://github.com/Neguin0/bot-modders.git```
Entre na pasta
```bash
cd bot-modders```
Instale os módulos
```bash
npm install```
Execute o script
```bash
node index.js```
